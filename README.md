## Co przygotować przed zajęciami?

- Node.js LTS (zalecane: 20.19.6 lub nowszy)
- Visual Studio Code (lub inny edytor)
- Przeglądarka Google Chrome
- Terminal ( Git Bash / PowerShell / Terminal macOS/Linux)

## Node.js - instalacja

Jeśli masz już Node.js, upewnij się, że używasz wersji `20.19.6` lub nowszej.

Pobieranie:

- Windows: https://nodejs.org/dist/v20.19.6/node-v20.19.6-x64.msi
- macOS: https://nodejs.org/dist/v20.19.6/node-v20.19.6.pkg
- Linux: https://nodejs.org/dist/v20.19.6/node-v20.19.6-linux-x64.tar.xz

Weryfikacja po instalacji:

```bash
node -v
npm -v
```

## Visual Studio Code - instalacja

Jeśli nie masz jeszcze edytora kodu, pobierz VS Code:
https://code.visualstudio.com/download
